================================================================================
ENVIRONMENT VARIABLES
================================================================================

Required for Email Sending:
  SMTP_USER          - Gmail address (e.g., hello@rezthegiant.com)
  SMTP_PASS          - Google App Password (16 chars, with spaces)
  SMTP_FROM          - From email address (same as SMTP_USER)

Optional (with defaults):
  PORT               - Server port (default: 5000)
  SMTP_HOST          - SMTP server (default: smtp.gmail.com)
  SMTP_PORT          - SMTP port (default: 587)
  DB_PATH            - SQLite path (default: /tmp/agency_outreach.db)
  SETTINGS_PATH      - Settings JSON (default: /tmp/agency_settings.json)
